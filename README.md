# Hyprland Dotfiles

Catppuccin Mocha pastel theme for Arch Linux + Hyprland + NVIDIA

**Hardware:** Maibenben X525 | i5-12400 | RTX 4060 Mobile | 144Hz

## What's included

- **Hyprland** — tiling WM with animations, blur, rounded corners
- **Waybar** — top panel with workspaces, clock, battery, audio, network
- **Kitty** — terminal with Catppuccin colors
- **Fish** — shell with starship prompt
- **Wofi** — app launcher (Super+D)
- **Dunst** — notifications
- **Hyprlock** — lock screen with clock, date, blur
- **Hypridle** — auto-lock, auto-suspend
- **SDDM Sugar Candy** — login screen theme
- **cava** — audio visualizer
- **swww** — animated wallpapers

## Install

```bash
git clone https://github.com/YOUR_USERNAME/hyprland-config.git ~/hyprland-config
cd ~/hyprland-config
chmod +x install.sh
./install.sh
```

See [instruction.md](instruction.md) for the full Arch Linux installation guide.
